# MyTgBotWithLLM

# Telegram-бот с поддержкой контекста на основе локальной LLM

Telegram-бот, который использует локальную языковую модель через LM Studio для генерации ответов с учётом истории диалога. Бот запоминает контекст разговора для каждого пользователя отдельно.

## Описание

Бот реализует систему управления контекстом диалога:
- Сохраняет историю сообщений для каждого пользователя
- Передаёт полный контекст в языковую модель при каждом запросе
- Позволяет очистить историю диалога командой `/clear`
- Поддерживает независимые диалоги для разных пользователей

## Требования

- Python 3.9+
- LM Studio (для запуска локальной LLM)
- Языковая модель (рекомендуется Qwen2.5-1.5B-Instruct-GGUF или аналогичная)

## Установка

### 1. Клонирование репозитория

```bash
git clone https://github.com/your-username/telegram-llm-bot.git
cd telegram-llm-bot
```
### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```
### 3. Ключ
- Вставить ключ бота в файл config.py

## Запуск
- Запустить сервер в LM Studio
- Запустить код командой 
```bash
python bot.py
```
- Запустить бота в Telegram (имя бота: @nottoolargellmbot)
